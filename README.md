# Bridge for You
这是我的[个人网站][1]，基于 [Github Pages][2] 和 [Jekyll][3] 搭建。

 **Bridge for You** 这个名字，灵感来自于之前听到的一句话——“**如果河上有桥，为什么还有摸着石头过河**”，的确，如果我们在学习的过程中，总是能够找到那座“桥”，那在有限的生命里就可以做更多有意义的事了，这也是我写文章的愿景，希望我的每一篇文章，都能对你有所帮助。
